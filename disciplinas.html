Principais Disciplinas

Detalhamos abaixo algumas das disciplinas mais marcantes do curso, com seus principais conteúdos e competências desenvolvidas.

Algoritmos e Lógica de Programação

Base do raciocínio computacional. Estruturas sequenciais, condicionais, repetições, vetores e matrizes utilizando linguagem C ou Python.

Programação Orientada a Objetos

Classes, objetos, herança, polimorfismo e encapsulamento com Java. Fundamento para todo software moderno.

Estrutura de Dados

Listas, pilhas, filas, árvores, grafos e algoritmos de ordenação e busca.

Banco de Dados

Modelagem entidade-relacionamento, SQL, normalização, MySQL e PostgreSQL.

Desenvolvimento Web Front-end

HTML5, CSS3, JavaScript, design responsivo e frameworks como React.

Desenvolvimento Web Back-end

Node.js, PHP ou Python, criação de APIs REST, autenticação e integração com banco de dados.

Desenvolvimento Mobile

Aplicativos Android com Java/Kotlin ou multiplataforma com Flutter/React Native.

Engenharia de Software

Levantamento de requisitos, UML, metodologias ágeis (Scrum, Kanban) e ciclo de vida do software.

Redes de Computadores

Modelo OSI, TCP/IP, configuração de redes, roteamento e protocolos.

Qualidade e Teste de Software

Testes unitários, integração, TDD, ferramentas como JUnit e Selenium.

Gerência de Projetos

PMBOK, métodos ágeis, ferramentas como Jira, Trello, e gestão de equipes.

Segurança da Informação

Criptografia, autenticação, principais vulnerabilidades (OWASP) e boas práticas.

IFCE Campus Boa Viagem

Curso Superior em Análise e Desenvolvimento de Sistemas. Formando profissionais para a transformação digital do sertão central cearense.
