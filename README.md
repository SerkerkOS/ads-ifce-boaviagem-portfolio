# IFCE ADS Boa Viagem — Portfolio do Curso

Site institucional do curso superior de **Análise e Desenvolvimento de Sistemas (ADS)** do **IFCE Campus Boa Viagem**. Uma vitrine digital do curso, apresentando informações sobre grade curricular, disciplinas, projetos, tecnologias, mercado de trabalho e muito mais.

## Sobre o Projeto

Este portfólio tem como objetivo divulgar o curso de ADS do IFCE Boa Viagem para a comunidade, destacando:

- Estrutura do campus e do curso
- Grade curricular e disciplinas
- Projetos desenvolvidos pelos alunos
- Tecnologias estudadas ao longo da formação
- Oportunidades de estágio e extensão
- Depoimentos e eventos acadêmicos

## Tecnologias

- HTML5
- CSS3
- JavaScript (em desenvolvimento)

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/SerkerkOS/ads-ifce-boaviagem-portfolio.git
   ```
2. Abra o arquivo `index.html` no navegador.
