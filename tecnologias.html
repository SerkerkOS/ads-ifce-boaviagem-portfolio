Tecnologias Estudadas

O curso aborda um amplo conjunto de linguagens, frameworks e ferramentas alinhadas com o mercado de trabalho.

Linguagens de Programação

Python

Ideal para iniciantes, ciência de dados e automação.

Java

Orientação a objetos, aplicações corporativas e Android.

JavaScript

Linguagem da web — front-end e back-end (Node).

PHP

Desenvolvimento web com Laravel.

C

Base para algoritmos e estrutura de dados.

SQL

Linguagem para banco de dados relacionais.

Frameworks e Bibliotecas

Front-end

React, Vue.js, Bootstrap, Tailwind CSS

Back-end

Node.js (Express), Laravel, Django, Spring Boot

Mobile

Flutter, React Native, Android Studio

Banco de Dados

  • MySQL, PostgreSQL, SQLite
  • MongoDB (NoSQL)
  • Firebase

Ferramentas de Desenvolvimento

  • Git e GitHub — controle de versão
  • VS Code, IntelliJ, Eclipse — IDEs
  • Figma — design de interfaces
  • Docker — containers
  • Postman — testes de API
  • Jira, Trello — gestão de projetos
